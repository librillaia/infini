37. Opera y simplifica cuando sea posible (Paso a Paso):

(a) (5x - 10) / (x + 3) · (x^2 - 9) / (x - 2)
1. Factorizar los términos:
    5x - 10 = 5(x - 2)
    x^2 - 9 = (x - 3)(x + 3)
2. Sustituir los términos factorizados en la expresión:
    [5(x - 2)] / (x + 3) · [(x - 3)(x + 3)] / (x - 2)
3. Multiplicar las fracciones:
    [5(x - 2)(x - 3)(x + 3)] / [(x + 3)(x - 2)]
4. Simplificar cancelando términos comunes (x - 2) y (x + 3):
    5(x - 3)
Resultado final: 5x - 15
(b) (2x + 1) / (2x - 1) ÷ x^2 / (4x - 2)
1. Reemplazar la división por la multiplicación por el recíproco de la segunda fracción:
    (2x + 1) / (2x - 1) · (4x - 2) / x^2
2. Factorizar los términos:
    4x - 2 = 2(2x - 1)
3. Sustituir el término factorizado:
    (2x + 1) / (2x - 1) · [2(2x - 1)] / x^2
4. Multiplicar las fracciones:
    [ (2x + 1) · 2(2x - 1) ] / [ (2x - 1) · x^2 ]
5. Simplificar cancelando el término común (2x - 1):
    2(2x + 1) / x^2
6. Expandir el numerador:
Resultado final: (4x + 2) / x^2
(c) x / 3 · (2x + 1) / (x - 1)
1. Multiplicar los numeradores:
    x · (2x + 1) = 2x^2 + x
2. Multiplicar los denominadores:
    3 · (x - 1) = 3x - 3
3. Combinar los resultados en una sola fracción:
Resultado final: (2x^2 + x) / (3x - 3)
(d) 1 / (x - 1) ÷ (x + 1) / (3x)
1. Reemplazar la división por la multiplicación por el recíproco de la segunda fracción:
    1 / (x - 1) · (3x) / (x + 1)
2. Multiplicar los numeradores:
    1 · 3x = 3x
3. Multiplicar los denominadores:
    (x - 1) · (x + 1) = x^2 - 1 (diferencia de cuadrados)
4. Combinar los resultados en una sola fracción:
Resultado final: 3x / (x^2 - 1)
(e) 2 / (x - 1) ÷ x / (x + 1)
1. Reemplazar la división por la multiplicación por el recíproco de la segunda fracción:
    2 / (x - 1) · (x + 1) / x
2. Multiplicar los numeradores:
    2 · (x + 1) = 2x + 2
3. Multiplicar los denominadores:
    (x - 1) · x = x^2 - x
4. Combinar los resultados en una sola fracción:
Resultado final: (2x + 2) / (x^2 - x)
(f) 2x / (2x - 3) ÷ (x + 1) / (2x + 3)
1. Reemplazar la división por la multiplicación por el recíproco de la segunda fracción:
    2x / (2x - 3) · (2x + 3) / (x + 1)
2. Multiplicar los numeradores:
    2x · (2x + 3) = 4x^2 + 6x
3. Multiplicar los denominadores (expandir):
    (2x - 3) · (x + 1) = 2x(x) + 2x(1) - 3(x) - 3(1)
    = 2x^2 + 2x - 3x - 3
    = 2x^2 - x - 3
4. Combinar los resultados en una sola fracción:
Resultado final: (4x^2 + 6x) / (2x^2 - x - 3)
