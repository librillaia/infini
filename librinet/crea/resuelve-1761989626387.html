37. Opera y simplifica cuando sea posible:

(a) (5x - 10) / (x + 3) · (x^2 - 9) / (x - 2) = 5(x - 3) = 5x - 15
(b) (2x + 1) / (2x - 1) ÷ x^2 / (4x - 2) = (2x + 1) / (2x - 1) · (4x - 2) / x^2 = (2x + 1) / (2x - 1) · 2(2x - 1) / x^2 = 2(2x + 1) / x^2 = (4x + 2) / x^2
(c) x / 3 · (2x + 1) / (x - 1) = (x(2x + 1)) / (3(x - 1)) = (2x^2 + x) / (3x - 3)
(d) 1 / (x - 1) ÷ (x + 1) / (3x) = 1 / (x - 1) · (3x) / (x + 1) = 3x / ((x - 1)(x + 1)) = 3x / (x^2 - 1)
(e) 2 / (x - 1) ÷ x / (x + 1) = 2 / (x - 1) · (x + 1) / x = (2(x + 1)) / (x(x - 1)) = (2x + 2) / (x^2 - x)
(f) 2x / (2x - 3) ÷ (x + 1) / (2x + 3) = 2x / (2x - 3) · (2x + 3) / (x + 1) = (2x(2x + 3)) / ((2x - 3)(x + 1)) = (4x^2 + 6x) / (2x^2 + 2x - 3x - 3) = (4x^2 + 6x) / (2x^2 - x - 3)
