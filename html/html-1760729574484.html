Ejercicios de Química

1. Números Atómicos y Másicos (Aniones y Cationes)

  1. Ejercicio 1.1

    Un átomo tiene 20 protones, 20 neutrones y 18 electrones.
    a) ¿Cuál es su número atómico (Z)?
    b) ¿Cuál es su número másico (A)?
    c) ¿Es un anión, un catión o un átomo neutro? ¿Cuál es su carga neta?

  2. Ejercicio 1.2

    El ion S²⁻ (sulfuro) tiene un número atómico Z=16 y número másico A=32.
    a) ¿Cuántos protones tiene?
    b) ¿Cuántos neutrones tiene?
    c) ¿Cuántos electrones tiene?

  3. Ejercicio 1.3

    El ion Al³⁺ (aluminio) tiene 13 protones y 14 neutrones.
    a) ¿Cuál es su número atómico (Z)?
    b) ¿Cuál es su número másico (A)?
    c) ¿Cuántos electrones tiene?

  4. Ejercicio 1.4

    Un átomo tiene Z=17 y A=35. Su carga es -1.
    a) ¿Cuántos protones tiene?
    b) ¿Cuántos neutrones tiene?
    c) ¿Cuántos electrones tiene?

2. Configuraciones Electrónicas y Diagrama de Moeller

Orden de llenado de orbitales (Diagrama de Moeller):

  • 1s
  • 2s 2p
  • 3s 3p
  • 4s 3d 4p
  • 5s 4d 5p
  • 6s 4f 5d 6p
  • 7s 5f 6d 7p

    i) Grupo y Periodo en la tabla periódica.
    ii) El elemento y su nombre (si es un grupo especial, indicarlo).
    a) Z = 11 (Sodio)
    b) Z = 17 (Cloro)
    c) Z = 20 (Calcio)
    d) Z = 26 (Hierro)
    e) Z = 35 (Bromo)

  2. Ejercicio 2.2

    Escribe la configuración electrónica de los siguientes iones:
    a) O²⁻ (Z=8)
    b) K⁺ (Z=19)
    c) Fe³⁺ (Z=26)

  3. Ejercicio 2.3

    Un elemento tiene la configuración electrónica terminal 4s²3d¹⁰4p⁵.
    a) ¿Cuál es su número atómico (Z)?

3. Números Cuánticos (n, l, m_l, m_s)

  1. Ejercicio 3.1

    Para el último electrón de los siguientes elementos en estado fundamental:
    a) Cloro (Z=17)
    b) Titanio (Z=22)
    c) Bromo (Z=35)
    Determina los cuatro números cuánticos (n, l, m_l, m_s). Asume que el último electrón se aparea o se añade en el último orbital disponible con spin +1/2.

  2. Ejercicio 3.2

    ¿Cuáles son los posibles valores de los números cuánticos (n, l, m_l, m_s) para un electrón en un orbital 3p?

  3. Ejercicio 3.3

    Indica si los siguientes conjuntos de números cuánticos son posibles o imposibles y justifica tu respuesta:
    a) (3, 2, 0, +1/2)
    b) (2, 2, -1, -1/2)
    c) (4, 1, +2, +1/2)
    d) (1, 0, 0, -1/2)
    e) (0, 0, 0, +1/2)

  4. Ejercicio 3.4

    Describe un conjunto de números cuánticos para cada uno de los 3 electrones de un orbital 2p.

Soluciones

Solución 1.1

a) Z = número de protones = 20.

b) A = protones + neutrones = 20 + 20 = 40.

c) El número de protones (20) es diferente al número de electrones (18). Hay más protones que electrones, por lo tanto es un catión. La carga neta es +2 (20 protones - 18 electrones = +2).

Solución 1.2

a) Protones = Z = 16.

b) Neutrones = A - Z = 32 - 16 = 16.

c) Para un ion S²⁻, tiene 16 protones y una carga de -2, lo que significa que tiene 2 electrones más que protones. Electrones = 16 + 2 = 18.

Solución 1.3

a) Z = número de protones = 13.

b) A = protones + neutrones = 13 + 14 = 27.

c) Para un ion Al³⁺, tiene 13 protones y una carga de +3, lo que significa que tiene 3 electrones menos que protones. Electrones = 13 - 3 = 10.

Solución 1.4

a) Protones = Z = 17.

b) Neutrones = A - Z = 35 - 17 = 18.

c) Para un ion con Z=17 y carga -1, tiene 17 protones y 1 electrón más que protones. Electrones = 17 + 1 = 18.

Solución 2.1

a) Z = 11 (Na): 1s²2s²2p⁶3s¹
i) Periodo: 3, Grupo: 1 (por 3s¹)
ii) Sodio (Na), Grupo: Metales Alcalinos

b) Z = 17 (Cl): 1s²2s²2p⁶3s²3p⁵
i) Periodo: 3, Grupo: 17 (por 3s²3p⁵ -> 2+5=7 electrones de valencia en p -> Grupo 17)
ii) Cloro (Cl), Grupo: Halógenos

c) Z = 20 (Ca): 1s²2s²2p⁶3s²3p⁶4s²
i) Periodo: 4, Grupo: 2 (por 4s²)
ii) Calcio (Ca), Grupo: Metales Alcalinotérreos

d) Z = 26 (Fe): 1s²2s²2p⁶3s²3p⁶4s²3d⁶
i) Periodo: 4, Grupo: 8 (elementos de transición, 4s²3d⁶ -> 2+6=8)
ii) Hierro (Fe), Grupo: Metales de Transición

e) Z = 35 (Br): 1s²2s²2p⁶3s²3p⁶4s²3d¹⁰4p⁵
i) Periodo: 4, Grupo: 17 (por 4s²4p⁵ -> 2+5=7 electrones de valencia -> Grupo 17)
ii) Bromo (Br), Grupo: Halógenos

Solución 2.2

a) O²⁻ (Z=8): Tiene 8 protones y 8+2=10 electrones. 1s²2s²2p⁶

b) K⁺ (Z=19): Tiene 19 protones y 19-1=18 electrones. 1s²2s²2p⁶3s²3p⁶

c) Fe³⁺ (Z=26): Tiene 26 protones y 26-3=23 electrones. La configuración del Fe neutro es [Ar]4s²3d⁶. Para formar el ion, se pierden primero los electrones del nivel más externo (4s), y luego los de 3d. Por lo tanto, se pierden 2 electrones del 4s y 1 del 3d.
1s²2s²2p⁶3s²3p⁶3d⁵ o [Ar]3d⁵

Solución 2.3

a) Configuración completa: 1s²2s²2p⁶3s²3p⁶4s²3d¹⁰4p⁵. Suma de electrones = 2+2+6+2+6+2+10+5 = 35. Por lo tanto, Z = 35.

b) El nivel más alto de energía principal (n) es 4, por lo tanto, Periodo: 4.
Electrones de valencia = 4s²4p⁵ = 2+5 = 7. Grupo: 17.

c) El elemento es Bromo (Br).

d) El grupo es Halógenos.

Solución 3.1

a) Cloro (Z=17): Configuración 1s²2s²2p⁶3s²3p⁵. El último electrón está en 3p⁵.
n = 3 (tercer nivel de energía)
l = 1 (orbital p)
m_l = -1, 0, +1 (para 3p⁵: El 3p tiene 3 orbitales. Se llenan 3 con spin +1/2, luego 2 se aparean con spin -1/2. El quinto electrón será el segundo que se aparea, en el primer orbital, o el que tiene m_l = 0 con spin -1/2 si se llenan de -1 a +1)
Asumiendo el llenado de -1 a +1 para m_l, los 5 electrones de 3p son:
(3, 1, -1, +1/2)
(3, 1, 0, +1/2)
(3, 1, +1, +1/2)
(3, 1, -1, -1/2)
(3, 1, 0, -1/2) <- Este es el último electrón
Entonces, (n, l, m_l, m_s) = (3, 1, 0, -1/2)

b) Titanio (Z=22): Configuración 1s²2s²2p⁶3s²3p⁶4s²3d². El último electrón está en 3d².
n = 3 (tercer nivel de energía)
l = 2 (orbital d)
m_l = -2, -1, 0, +1, +2 (para 3d²: los electrones se distribuyen con spin paralelo primero)
Los 2 electrones de 3d son:
(3, 2, -2, +1/2)
(3, 2, -1, +1/2) <- Este es el último electrón
Entonces, (n, l, m_l, m_s) = (3, 2, -1, +1/2)

c) Bromo (Z=35): Configuración 1s²2s²2p⁶3s²3p⁶4s²3d¹⁰4p⁵. El último electrón está en 4p⁵.
n = 4 (cuarto nivel de energía)
l = 1 (orbital p)
m_l = -1, 0, +1 (para 4p⁵: se llenan 3 con spin +1/2, luego 2 se aparean con spin -1/2)
Los 5 electrones de 4p son:
(4, 1, -1, +1/2)
(4, 1, 0, +1/2)
(4, 1, +1, +1/2)
(4, 1, -1, -1/2)
(4, 1, 0, -1/2) <- Este es el último electrón
Entonces, (n, l, m_l, m_s) = (4, 1, 0, -1/2)

Solución 3.2

Para un orbital 3p:

  • n = 3 (El número principal es 3)
  • l = 1 (Para orbitales 'p', l siempre es 1)
  • m_l = -1, 0, +1 (Para l=1, m_l puede ser -l, ..., 0, ..., +l)
  • m_s = +1/2 o -1/2 (El spin magnético siempre puede tener dos valores)

Conjuntos posibles para un electrón en un orbital 3p:

  • (3, 1, -1, +1/2)
  • (3, 1, -1, -1/2)
  • (3, 1, 0, +1/2)
  • (3, 1, 0, -1/2)
  • (3, 1, +1, +1/2)
  • (3, 1, +1, -1/2)

Solución 3.3

a) (3, 2, 0, +1/2) - **Posible.** n=3, l=2 (orbital d, posible si n=3). Para l=2, m_l puede ser -2, -1, 0, +1, +2 (0 es posible). m_s es siempre +1/2 o -1/2.

b) (2, 2, -1, -1/2) - **Imposible.** Si n=2, los valores posibles para l son 0 y 1. l=2 no es posible cuando n=2 (l debe ser menor que n).

c) (4, 1, +2, +1/2) - **Imposible.** Si l=1, los valores posibles para m_l son -1, 0, +1. m_l = +2 no es posible cuando l=1 (m_l debe ser entre -l y +l).

d) (1, 0, 0, -1/2) - **Posible.** n=1, l=0 (orbital s, posible si n=1). Para l=0, m_l solo puede ser 0 (0 es posible). m_s es siempre +1/2 o -1/2.

e) (0, 0, 0, +1/2) - **Imposible.** n (el número cuántico principal) debe ser un entero positivo (1, 2, 3,...). n=0 no es posible.

Solución 3.4

Un orbital 2p significa n=2 y l=1. Los posibles valores de m_l son -1, 0, +1. Cada uno de estos suborbitales puede albergar 2 electrones con spines opuestos.

Si hay 3 electrones en el orbital 2p, por la regla de Hund, ocuparán suborbitales diferentes con spins paralelos (si es posible).

Ejemplo de un conjunto de números cuánticos para los 3 electrones de un orbital 2p:

  • Electrón 1: (2, 1, -1, +1/2)
  • Electrón 2: (2, 1, 0, +1/2)
  • Electrón 3: (2, 1, +1, +1/2)

Otras combinaciones con m_s = -1/2 también serían válidas si el llenado es diferente, pero la primera opción con spins paralelos es la más común para electrones desapareados en orbitales degenerados.
